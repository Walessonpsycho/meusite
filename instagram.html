Como obter o Access Token

      

        A API do Instagram exige autenticação. Siga o passo a passo abaixo para obter seu token gratuito.
      


      

        ⚠️
        

          Token de Longa Duração: O token básico expira em 1 hora. Você precisa gerar um Long-lived token (válido por 60 dias) e renová-lo periodicamente. O processo abaixo gera o token longo automaticamente.
        

      


      

        

          

            
1

            

              
Criar conta de desenvolvedor Meta

              
Acesse developers.facebook.com e faça login com sua conta do Facebook vinculada ao Instagram profissional.

            

          

        

        

          

            
2

            

              
Criar um App

              
Clique em Meus Apps → Criar App. Selecione "Consumidor" ou "Empresa" como tipo. Dê um nome ao app (ex: "WISE Feed").

            

          

        

        

          

            
3

            

              
Adicionar produto Instagram Basic Display

              
No painel do app, localize Produtos → procure "Instagram Basic Display" → clique em Configurar.

            

          

        

        

          

            
4

            

              
Adicionar usuário de teste

              
Em Funções → Usuários de teste do Instagram, adicione o perfil @psicologohomens_walessoncastro e aceite o convite no Instagram.

            

          

        

        

          

            
5

            

              
Gerar o Access Token

              
Em Instagram Basic Display → Tokens de acesso de usuário, clique em "Gerar Token" ao lado do seu usuário. Copie o token gerado e cole na aba "Configurar".

            

          

        

        

          💡
          

            Alternativa mais simples: Use o serviço Embedsocial, Elfsight ou Later — todos oferecem planos gratuitos que geram um widget de feed pronto para colar no site, sem precisar configurar API.
